Project

2MEAN is a MEAN Stack (MongoDB, Express.js, Angular, Node.js) starter repository featuring cutting edge technology such as Angular 2 with universal rendering. This project aims to cut down on the time it takes to setup an Angular 2 project with all the necessary tooling for TypeScript.

Role

Built as a solo project I researched and programmed the front-end, server and configurations for the tooling needed to build the project.
